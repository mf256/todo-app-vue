## Setup

How to run this project.

1. Clone this repo

2. To run, go to project folder and run

`$ npm install`

3. Now start dev server by running -

`$ npm run serve`

4. visit - http://localhost:8080/

To create production ready codes -

`$ npm run build`

for more commands refer `package.json`
